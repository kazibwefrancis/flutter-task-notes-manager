## Description

This Flutter application is a task and notes manager that allows users to create, view, and manage their tasks. Features include:

- Adding new tasks with title, priority, and description
- Viewing a list of all tasks
- Switching between light and dark themes
- Persistent storage using SQLite database
- Theme preference saved using SharedPreferences

## How to Run the Project

1. Ensure you have Flutter installed on your machine.
2. Clone or download this repository.
3. Navigate to the project directory.
4. Run `flutter pub get` to install dependencies.
5. Run `flutter run` to start the application on a connected device or emulator.

## Features

- **Home Screen:** Displays a list of tasks with a welcome message and theme toggle.
- **Add Task Screen:** Form to add new tasks with validation.
- **Theme Switching:** Toggle between light and dark modes, preference is saved.
- **Database Integration:** Tasks are stored locally using SQFLite.
